HexoPress 是一款专为 Hexo 博客打造的桌面编辑器，基于 Electron 构建，提供可视化的文章管理、编辑和媒体资源管理功能。如果你已经有一个 Hexo 博客，HexoPress 可以让你告别命令行，用图形界面来管理博客的一切。

## 安装

从 HexoPress 的 GitHub Release 页面下载对应你操作系统的安装包：

- macOS：下载 `.dmg` 文件，拖拽到 Applications 即可
- Windows：下载 `.exe` 安装包，双击安装
- Linux：下载 `.AppImage` 或 `.deb` 包

## 首次启动

启动 HexoPress 后，你会看到一个设置页面，要求你选择一个 Hexo 博客的目录。

"插图：请插入首次启动时的 Setup 设置页面截图，显示选择目录的按钮"

点击「选择目录」按钮，在弹出的文件选择器中，找到你本地的 Hexo 博客根目录（就是包含 `_config.yml`、`source`、`themes` 等文件夹的那个目录），选中它。

HexoPress 会自动检测这个目录是否是一个合法的 Hexo 博客。如果检测通过，就会开始加载博客数据，包括文章、分类、标签和媒体资源。

## 加载完成

数据加载完成后，你会进入 HexoPress 的主界面——仪表盘（Dashboard）。在这里你可以看到博客的概览信息，包括文章数量、写作热力图、最近发布的文章等。

"插图：请插入加载完成后的 Dashboard 仪表盘全貌截图，展示左侧导航栏和主内容区"

左侧的导航栏提供了以下入口：

- **仪表盘**：博客概览和统计
- **文章列表**：管理所有文章和草稿
- **分类**：查看和管理文章分类
- **标签**：查看和管理文章标签
- **媒体库**：管理博客中的图片和文件
- **偏好设置**：配置 HexoPress 的各项选项

到这里，你已经成功地将 HexoPress 连接到了你的 Hexo 博客。接下来可以继续阅读后续的教程，了解每个功能的详细用法。
